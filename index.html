Web Development/CRM Projects

Spur of the Moment - Concert Finder

First project from the Rutgers Coding Bootcamp Spring 2016 class. A geolocation based concert finder that allows you to search for an artist, and find concerts by similar artists nearby. Created using HTML, CSS, Booststrap, Javascript, jQuery, Google Maps API, BandsInTown API, and AJAX.

Cross Sport Fantasy Football

Second project from RUCB Spring 2016. A prototype model for cross-sport fantasy sports, allowing players to create a team of players across various sports, with weighted point values for equitable scoring. This presentation focuses on the back-end concepts as opposed to client-facing design, as teh front-end was still in development at the time of filming. Created using Handlebars, Express, Node.js, NPM, Javascript, FantasyData API, ORM/MVC structures, SQL, and Ajax.
